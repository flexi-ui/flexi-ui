# Layout

FlexiUI provides flexible layout utilities built on Tailwind CSS to help you create responsive designs.

## Container

Use container utilities for responsive layouts:

```tsx
<div className="container mx-auto px-4">
  {/* Your content */}
</div>
```

## Grid System

Create grid layouts with Tailwind's grid utilities:

```tsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <div>Column 1</div>
  <div>Column 2</div>
  <div>Column 3</div>
</div>
```

## Flexbox Layouts

Use flexbox for flexible layouts:

```tsx
<div className="flex items-center justify-between">
  <div>Left</div>
  <div>Right</div>
</div>
```

## Spacing

FlexiUI uses Tailwind's spacing scale:

```tsx
<div className="p-4">Padding</div>
<div className="m-4">Margin</div>
<div className="space-y-4">
  <div>Item 1</div>
  <div>Item 2</div>
</div>
```

## Responsive Design

Use responsive breakpoints:

```tsx
<div className="w-full md:w-1/2 lg:w-1/3">
  Responsive width
</div>
```

## Max Width

Control maximum width:

```tsx
<div className="max-w-7xl mx-auto">
  {/* Content */}
</div>
```

## Aspect Ratio

Maintain aspect ratios:

```tsx
<div className="aspect-video">
  <img src="image.jpg" alt="16:9 aspect ratio" />
</div>
```

## Stack Layout

Create vertical stacks with consistent spacing:

```tsx
<div className="space-y-4">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

## Center Content

Center content horizontally and vertically:

```tsx
<div className="flex items-center justify-center min-h-screen">
  <div>Centered content</div>
</div>
```

## Sidebar Layout

Create sidebar layouts:

```tsx
<div className="flex">
  <aside className="w-64">Sidebar</aside>
  <main className="flex-1">Main content</main>
</div>
```
