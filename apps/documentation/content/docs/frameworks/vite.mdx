# Vite

FlexiUI works with any Vite-based React project (`create-vite`, `tanstack-router`, custom setups). The setup is a Tailwind plugin and one CSS import.

## 1. Create the project

```bash
pnpm create vite@latest my-app -- --template react-ts
cd my-app
pnpm install
```

## 2. Install Tailwind v4 + FlexiUI

```bash
pnpm add -D tailwindcss @tailwindcss/vite
pnpm add @flexi-ui/styles @flexi-ui/button
```

## 3. Register the Tailwind Vite plugin

```ts
// vite.config.ts
import { defineConfig } from 'vite'
import react from '@vitejs/plugin-react'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [react(), tailwindcss()],
})
```

## 4. Create the stylesheet

```css
/* src/index.css */
@import '@flexi-ui/styles/css';

@custom-variant dark (&:where(.dark, .dark *));

:root {
  --background: oklch(1 0 0);
  --foreground: oklch(0.145 0 0);
}

.dark {
  --background: oklch(0.145 0 0);
  --foreground: oklch(0.985 0 0);
}
```

Import it once at your entry point:

```tsx
// src/main.tsx
import './index.css'
```

## 5. Use a component

```tsx
// src/App.tsx
import { Button } from '@flexi-ui/button'

export default function App() {
  return (
    <div className="flex min-h-screen items-center justify-center">
      <Button>Hello FlexiUI</Button>
    </div>
  )
}
```

Run `pnpm dev` and you're set.

## Client-side routing

If you're using React Router or TanStack Router, wrap your app in React Aria's `RouterProvider` so FlexiUI's `<Link>` uses client-side navigation. See the [Routing guide](/docs/guide/routing) for framework-specific snippets.

## Dark mode

There's no theme provider in FlexiUI. To toggle dark mode, add/remove a `dark` class on `<html>` (or any ancestor) — the styles package scopes its dark-mode rules with `@custom-variant dark (&:where(.dark, .dark *))`.

A one-file helper:

```tsx
// src/use-theme.ts
import { useEffect, useState } from 'react'

type Theme = 'light' | 'dark'

export function useTheme() {
  const [theme, setTheme] = useState<Theme>(
    () => (localStorage.getItem('theme') as Theme) ?? 'light',
  )

  useEffect(() => {
    document.documentElement.classList.toggle('dark', theme === 'dark')
    localStorage.setItem('theme', theme)
  }, [theme])

  return { theme, setTheme }
}
```
