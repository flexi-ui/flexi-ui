# Remix

FlexiUI works with Remix (and its successor, React Router v7 in framework mode). The setup is one Tailwind plugin line and one CSS import.

## 1. Create or open a project

```bash
pnpm create remix@latest my-app
cd my-app
```

## 2. Install Tailwind v4 and FlexiUI

```bash
pnpm add -D tailwindcss @tailwindcss/vite
pnpm add @flexi-ui/styles @flexi-ui/button
```

## 3. Register the Tailwind Vite plugin

```ts
// vite.config.ts
import { vitePlugin as remix } from '@remix-run/dev'
import { defineConfig } from 'vite'
import tailwindcss from '@tailwindcss/vite'

export default defineConfig({
  plugins: [remix(), tailwindcss()],
})
```

## 4. Create the stylesheet

```css
/* app/tailwind.css */
@import '@flexi-ui/styles/css';

@custom-variant dark (&:where(.dark, .dark *));

:root {
  --background: oklch(1 0 0);
  --foreground: oklch(0.145 0 0);
}

.dark {
  --background: oklch(0.145 0 0);
  --foreground: oklch(0.985 0 0);
}
```

Link it from `app/root.tsx`:

```tsx
// app/root.tsx
import type { LinksFunction } from '@remix-run/node'
import tailwindHref from './tailwind.css?url'

export const links: LinksFunction = () => [{ rel: 'stylesheet', href: tailwindHref }]

export default function App() {
  return (
    <html lang="en">
      <head>
        <Meta />
        <Links />
      </head>
      <body>
        <Outlet />
        <Scripts />
      </body>
    </html>
  )
}
```

## 5. Use a component

```tsx
// app/routes/_index.tsx
import { Button } from '@flexi-ui/button'

export default function Index() {
  return (
    <main className="flex min-h-screen items-center justify-center">
      <Button>Hello FlexiUI</Button>
    </main>
  )
}
```

## Client-side routing

Wrap your outlet in React Aria's `RouterProvider` using Remix's `useNavigate` and `useHref`. The [Routing guide](/docs/guide/routing) has the full snippet — with it, `<Link>` from `@flexi-ui/link` navigates through the Remix router instead of the browser.
