# Spinner

`Spinner` is a loading indicator for asynchronous work. It renders an SVG with a subtle gradient, applies a `role="status"` so assistive tech knows work is in progress, and accepts an optional `label` for the status text.

## Installation

```bash
pnpm add @flexi-ui/spinner @flexi-ui/styles
```

## Import

```tsx
import { Spinner } from '@flexi-ui/spinner'
```

Seven colors — five semantic, plus `current` (inherits from the parent's text color) and `white` (for use on dark overlays).

### `color="current"`

Inherit the parent's text color — useful when dropping a spinner into a colored container or a `Button`.

<Sandbox code={`<div className="flex items-center gap-3 text-danger">
  <Spinner color="current" />
  <span className="text-sm">Retrying…</span>
</div>`}>
  <div className="flex items-center gap-3 text-danger">
    <Spinner color="current" />
    <span className="text-sm">Retrying…</span>
  </div>
</Sandbox>

## Label

`label` renders screen-reader-accessible status text beside the spinner. Use it whenever the spinner is the only indicator that something is loading.

<Sandbox code={`<Spinner label="Loading your projects…" />`}>
  <Spinner label="Loading your projects…" />
</Sandbox>

## Inside a Button

A common pattern: swap a button's label for a spinner while a request is in flight. `color="current"` matches the button's text color automatically.

<Sandbox code={`<div className="flex gap-3">
  <Button variant="primary" isDisabled>
    <Spinner size="sm" color="current" /> Saving…
  </Button>
  <Button variant="bordered" isDisabled>
    <Spinner size="sm" color="current" /> Retrying
  </Button>
</div>`}>
  <div className="flex gap-3">
    <Button variant="primary" isDisabled>
      <Spinner size="sm" color="current" /> Saving…
    </Button>
    <Button variant="bordered" isDisabled>
      <Spinner size="sm" color="current" /> Retrying
    </Button>
  </div>
</Sandbox>

## API Reference

### Spinner

| Prop    | Type                                                                                            | Default     | Description                                                |
| ------- | ----------------------------------------------------------------------------------------------- | ----------- | ---------------------------------------------------------- |
| `size`  | `'sm' \| 'md' \| 'lg'`                                                                          | `'md'`      | Visual size.                                               |
| `color` | `'current' \| 'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger' \| 'white'`        | `'primary'` | Stroke color. `current` inherits the parent's text color.  |
| `label` | `string`                                                                                        | —           | Status text announced to screen readers and rendered beside the indicator. |

Extends native `svg` props (omits `display`, `opacity`, `color`, `size` — these are replaced by variant props).

## Accessibility

- Wrapped in a `<span role="status">` so assistive tech announces the loading state when the spinner appears.
- If the spinner is the only loading signal, always pass `label` — a silent spinner is invisible to screen readers.
- For long-running loads, consider pairing the spinner with a visible progress message so sighted users know what's being loaded.
