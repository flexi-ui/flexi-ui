# Avatar

`Avatar` renders a user's profile image with graceful fallbacks. If the image fails to load or isn't provided, it can fall back to a `name`-derived initials badge or any custom node.

## Installation

```bash
pnpm add @flexi-ui/avatar @flexi-ui/styles
```

## Import

```tsx
import { Avatar } from '@flexi-ui/avatar'
```

## Graceful image failure

If the image URL 404s or errors, Avatar silently falls back to `fallback` or `name`-initials — no broken icon flash.

```tsx
<Avatar src="/broken.png" name="Alex Morgan" />
```

## API Reference

### Avatar (Root)

| Prop         | Type                                  | Default   | Description                                             |
| ------------ | ------------------------------------- | --------- | ------------------------------------------------------- |
| `src`        | `string`                              | —         | Image URL.                                              |
| `alt`        | `string`                              | `name`    | Alt text for the image. Falls back to `name`.           |
| `name`       | `string`                              | —         | Used to derive initials when the image is unavailable.  |
| `fallback`   | `ReactNode`                           | —         | Custom node rendered when there's no image.             |
| `size`       | `'sm' \| 'md' \| 'lg' \| 'xl'`        | `'md'`    | Visual size.                                            |
| `isBordered` | `boolean`                             | `false`   | Adds a ring around the avatar.                          |

Extends native `span` props (omits `size`).

### Fallback priority

1. `src` and image loads → renders `<img>`
2. Else if `fallback` is provided → renders it
3. Else if `name` is provided → renders derived initials
4. Else → empty

## Accessibility

- The underlying `<img>` uses `alt` (falling back to `name`), so screen readers announce who the avatar represents.
- When displaying initials or a decorative fallback, provide a `name` so the avatar still has semantic meaning in context.
- Avatars are non-interactive by default. Wrap one in a `Link` or `button` to make it actionable.
