# Badge

`Badge` is a small inline label used for counts, tags, and status markers. It supports four visual variants (solid, flat, bordered, dot) and five semantic colors.

## Installation

```bash
pnpm add @flexi-ui/badge @flexi-ui/styles
```

## Import

```tsx
import { Badge } from '@flexi-ui/badge'
```

## API Reference

### Badge (Root)

| Prop      | Type                                                            | Default     | Description            |
| --------- | --------------------------------------------------------------- | ----------- | ---------------------- |
| `variant` | `'solid' \| 'flat' \| 'bordered' \| 'dot'`                      | `'solid'`   | Visual style.          |
| `color`   | `'default' \| 'accent' \| 'success' \| 'warning' \| 'danger'`   | `'default'` | Semantic color.        |
| `size`    | `'sm' \| 'md' \| 'lg'`                                          | `'md'`      | Visual size.           |
| `children`| `ReactNode`                                                     | —           | Label content.         |

Extends native `span` props (omits `color`).

## Accessibility

- Badges are decorative by default. If the badge conveys information not available elsewhere (e.g. "3 unread"), include it in the button's `aria-label` or an adjacent `VisuallyHidden` span.
- The `dot` variant's colored dot is purely visual — always pair it with a text label for screen readers.
