# Alert

`Alert` surfaces contextual feedback to the user — status updates, warnings, errors, or confirmations — with a compound API that lets you compose an icon, title, description, and close button.

The root renders a `div` with `role="alert"`, so assistive tech announces the contents when it appears.

## Installation

```bash
pnpm add @flexi-ui/alert @flexi-ui/styles
```

## Import

```tsx
import { Alert } from '@flexi-ui/alert'
```

## Anatomy

```tsx
<Alert>
  <Alert.Icon />
  <Alert.Content>
    <Alert.Title />
    <Alert.Description />
  </Alert.Content>
  <Alert.Close />
</Alert>
```

| Part            | Element | Purpose                                             |
| --------------- | ------- | --------------------------------------------------- |
| `Alert`         | `div`   | Root container. Applies variant + `role="alert"`.   |
| `Alert.Icon`    | `span`  | Leading icon slot.                                  |
| `Alert.Content` | `div`   | Wraps the title and description.                    |
| `Alert.Title`   | `h5`    | Short headline.                                     |
| `Alert.Description` | `p` | Longer supporting copy.                             |
| `Alert.Close`   | `button`| Dismiss button. Wire your own `onClick` to hide it. |

## Dismissible

Wire an `onClick` handler on `Alert.Close` and control visibility in the parent.

```tsx
'use client'

import { useState } from 'react'
import { Alert } from '@flexi-ui/alert'

export function DismissibleAlert() {
  const [open, setOpen] = useState(true)
  if (!open) return null

  return (
    <Alert variant="success">
      <Alert.Content>
        <Alert.Title>Changes saved</Alert.Title>
        <Alert.Description>Your profile has been updated.</Alert.Description>
      </Alert.Content>
      <Alert.Close onClick={() => setOpen(false)}>
        <span aria-hidden>×</span>
      </Alert.Close>
    </Alert>
  )
}
```

## API Reference

### Alert (Root)

| Prop       | Type                                                            | Default     | Description                                      |
| ---------- | --------------------------------------------------------------- | ----------- | ------------------------------------------------ |
| `variant`  | `'default' \| 'accent' \| 'success' \| 'warning' \| 'danger'`   | `'default'` | Status tone.                                     |
| `isFlat`   | `boolean`                                                       | `false`     | Use the lighter flat background.                 |
| `children` | `ReactNode`                                                     | —           | Icon, content, and close parts.                  |

Extends native `div` props.

### Alert.Icon / Alert.Content / Alert.Title / Alert.Description

Each accepts the native props of its underlying element (`span`, `div`, `h5`, `p`) plus `className` and `children`.

### Alert.Close

Renders a `<button type="button" aria-label="Close">`. Forwards all native button props including `onClick`.

## Accessibility

- Root sets `role="alert"`, which announces new alerts to screen readers as they appear.
- `Alert.Close` ships with `aria-label="Close"`; override it if your UI needs a different label.
- When an alert auto-dismisses, prefer unmounting the component rather than hiding it with CSS so the announcement semantics remain consistent.
