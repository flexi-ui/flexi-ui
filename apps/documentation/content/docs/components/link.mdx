# Link

`Link` is a styled anchor element built on React Aria Components' `Link` primitive. It carries keyboard, focus, and press handling from React Aria, and adds colors, sizes, underline strategies, and an external-link helper on top.

## Installation

```bash
pnpm add @flexi-ui/link @flexi-ui/styles
```

## Import

```tsx
import { Link } from '@flexi-ui/link'
```

## API Reference

### Link

| Prop             | Type                                                                              | Default     | Description                                                              |
| ---------------- | --------------------------------------------------------------------------------- | ----------- | ------------------------------------------------------------------------ |
| `color`          | `'foreground' \| 'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger'`  | `'primary'` | Text color.                                                              |
| `size`           | `'sm' \| 'md' \| 'lg'`                                                            | `'md'`      | Visual size.                                                             |
| `underline`      | `'none' \| 'hover' \| 'always' \| 'active'`                                       | `'none'`    | Underline strategy.                                                      |
| `isBlock`        | `boolean`                                                                         | `false`     | Pill-shaped hover target for list-like layouts.                          |
| `isDisabled`     | `boolean`                                                                         | `false`     | Disables the link.                                                       |
| `isExternal`     | `boolean`                                                                         | `false`     | Adds `target="_blank"` and `rel="noopener noreferrer"`.                  |
| `showAnchorIcon` | `boolean`                                                                         | `false`     | Appends an external-link icon.                                           |
| `anchorIcon`     | `ReactNode`                                                                       | built-in    | Replace the default anchor icon.                                         |

Extends all props of React Aria's `Link` â€” including `href`, `onPress`, and render-prop `children`.

## Accessibility

- Renders a real `<a>` element when `href` is provided, so screen readers and routers treat it like any other link.
- `isExternal` adds `rel="noopener noreferrer"` so the opened page can't hijack the `window.opener` reference.
- When `showAnchorIcon` is on, the default icon is `aria-hidden` â€” the link's own text remains the accessible name.
- For links that *look* like buttons (e.g. primary CTA), consider whether a `Button` is more semantically honest.
